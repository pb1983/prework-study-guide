HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Semenatic Elements describe their purpose in it's respective tag.
  • Non-semantic elements do not describe their intended use as clearly as Semantic Elements.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A Class selector allows you to apply style to specific sections of code instead of all elements in the .html file.
  • Standard practice is to link an external .css file to your .html document for optimal flexiblity.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • A Pull notifies others about changes that have been pushed to a branch.
  • A Push commits the changes to a branch.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • "If else" statements allow the code to execute a specific condition and elminate unwanted actions.
  • A "for loop" can execute a black of code several times.
  • An "array" allows you to store multiple objects in one variable.
  • A "funtcion" can consolidate numerous lines of code into a single task.
